## 소개
실무에서 팀에 업무를 파악하기 위해 이벤트를 JSON에 저장하여, 

연도, 월, 팀원, 카테고리, 키워드 등 시각적으로 업무량을 확인 할 수 있도록 구현하였습니다. 

## 기능
- 반응형 UI
- 다크모드 
- 차트모드
- 카테고리별 비교
